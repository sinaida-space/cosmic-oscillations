# Sinaida AI Pitch Deck Generator

A production-ready, single-page web application that transforms startup ideas into executive-level investor slide decks using the Hugging Face Inference API.

## Features
- **Swiss International Style**: Strict modular grid design for a premium, minimalist aesthetic.
- **12-Step Strategic Framework**: Guides users through 12 critical questions for investment readiness.
- **AI-Powered Generation**: Uses Mistral-7B via Hugging Face to refine raw answers into sharp executive summaries.
- **Client-Side PPTX Generation**: Downloads a branded, 16:9 PowerPoint deck directly from the browser.
- **Three.js Hero**: Interactive, high-performance parallax background.
- **GDPR Compliant**: No server-side storage, no tracking, pure client-side processing.

## Quick Start
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. (Optional) Deploy to GitHub Pages:
   - Upload `index.html`, `.nojekyll`, and `README.md`.
   - Enable GitHub Pages in settings.

## Technical Details
- **Core**: HTML5, Vanilla CSS3 (Custom Properties), Vanilla JS (ES6+).
- **Libraries**: Three.js (Animation), PptxGenJS (PowerPoint).
- **Inference**: Mistral-7B-Instruct-v0.2 (Hugging Face Public API).
